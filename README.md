# Expense Tracker
A Flask-based web application to track daily expenses with filtering, charts, and real-time balance calculation.
Built as a beginner project to learn Flask and full-stack development.

## Features
- Add, edit, delete expenses  
- Filter by date and category  
- Visualize spending using charts (Chart.js)  
- Dynamic salary-based balance tracking  
- Export data as CSV  

## Live Demo
https://expense-tracker-1-sdmb.onrender.com/

## Tech Stack
- Flask (Python)  
- SQLite  
- HTML, Tailwind CSS  
- Chart.js

```bash

cd expense-tracker
pip install -r requirements.txt
python app.py
```
